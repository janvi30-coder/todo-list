📌 Project Overview

This is a simple console-based To-Do List application built using Python. The program allows users to manage their daily tasks through a menu-driven interface.

✨ Features

* Add a new task
* View all tasks
* Remove a task
* Exit the program

🛠️ Concepts Used

* Python Lists
* Loops (`for` and `while`)
* Conditional Statements (`if`, `elif`, `else`)
* User Input (`input()`)
* List Methods (`append()`, `remove()`)

▶️ How to Run

1. Download or clone this repository.
2. Open the project in Python.
3. Run `todo_list.py`.
4. Follow the on-screen menu.

📚 What I Learned

* How to store data using lists.
* How to add and remove items from a list.
* How to use loops to display tasks.
* How to build a simple menu-driven Python program.
